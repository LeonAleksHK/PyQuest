# PyQuest — 2D платформер для обучения Python

Образовательная игра: прыгай по платформам, собирай баги и решай задачи по Python!

## Быстрый старт

```bash
pip install pygame
python main.py
```

## Управление

| Клавиша | Действие |
|---------|----------|
| ← → / A D | Движение |
| ↑ / W / Пробел | Прыжок |
| ESC | Закрыть квест / Выход |

## Структура игры

```
pyquest/
├── main.py      # Основной игровой цикл, физика, рендер
├── quests.py    # База данных квестов (14 заданий по Python)
├── ui.py        # UI-компоненты (QuestPanel, HUD, экраны)
└── README.md
```

## Темы квестов

### Уровень 1 — Базовый синтаксис
- Бесконечный цикл (while без счётчика)
- Неверный тип (input vs int)
- Отступы (IndentationError)

### Уровень 2 — Списки
- IndexError (индекс за границей)
- Изменение списка при итерации
- Копия vs ссылка

### Уровень 3 — Функции
- Область видимости (scope)
- Изменяемый аргумент по умолчанию
- lambda и замыкание (closure)

### Уровень 4 — Ошибки
- try/except (ZeroDivisionError)
- Сравнение с None (is vs ==)

### Уровень 5 — ООП и продвинутое
- Забытый self в методе класса
- f-строки (форматирование)
- Словарь: .get() vs []

## Как добавить свой квест

В файле `quests.py` добавь новый элемент в список `QUESTS`:

```python
{
    "name": "Название квеста",
    "topic": "Тема",
    "xp": 70,
    "desc": "Описание задачи\n(можно в две строки)",
    "code": [
        ("строка кода", "ok"),    # ok = корректная строка
        ("ошибочная",   "bug"),   # bug = красная подсветка
        ("# комментарий", "hint"), # hint = жёлтый цвет
    ],
    "options": [
        {"text": "Правильный ответ", "correct": True},
        {"text": "Неверный вариант", "correct": False},
        {"text": "Ещё неверный",     "correct": False},
        {"text": "И ещё один",       "correct": False},
    ],
    "hint": "Текст подсказки\n(показывается по кнопке или после ошибки)",
    "explanation": "Объяснение правильного ответа\n(показывается после верного ответа)",
}
```
